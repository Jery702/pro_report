### 微信公众号模块描述

+ information -- 规章制度、河长动态、信息详情
+ event -- 曝光台、事件上报、我的事件、曝光台详情
+ photograph -- 随手拍、随手拍列表、我的随手拍、随手拍详情
+ reach -- 河道列表、河道详情、河道水质、一河一策
+ suggest -- 建言献策、满意度评价
+ user -- 个人信息
+ download -- app下载
+ skyBlue -- 蔚蓝天空
